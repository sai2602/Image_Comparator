1. In order to run the code, change the data_path variable in the code to the root directory of the images folder
2. With the code left as it is, standard filtering process takes place
3. TO play around with the filtering criteria, change the:
	CONTOUR_AREA: contour areas less than this number are ignored and are just considered as noise
	THRESHOLD_AREA: increasing this value implies strict filtering of images and as the value decreases, more lenient the filtering gets
